Dúvidas que podem surgir:

| Como posso obter um certificado?

Fazendo todos os exercícios, um projeto final e fazendo o teste final e tendo mais do que a media de acertos.

| Os certificados são gratuitos?

Sim! Todos os certificados emitidos pela LogicLab são 100% gratuitos.

| O certificado é válido oficialmente?

Os certificados são válidos como comprovante de participação e podem ser usados para horas complementares, dependendo da instituição.

| Como faço para conseguir meu certificado?

Após concluir o módulo, faça a avaliação final do nível de exercícios e depois coloque o link de seu projeto final de cada nível. Depois disso o sistema irá gerar seu certificado e você irá para uma página na Web com seu certificado e com as ferramentas do navegador baixar o certificado em PDF.

| Dicas e boas práticas quando preencher os campos para a emissão do certificado.

Dica 1: Verifique se o link do repositório do GitHub do seu projeto está correto, senão a emissão do certificado sairá errada.

Dica 2: Coloque seu nome completo. Se o nome completo for muito grande, coloque de forma abreviada.

Dica 3: Verifique se a data do dispositivo que você está acessando o site está correta, pois durante a emissão do certificado para mais precisão colocamos a data de emissão.

Dica 4: Nunca emita um certificado se não fez os exercícios. Lembre-se: se você está fazendo isso, não está só enganando o sistema, mas sim a você mesmo. O importante é o aprendizado. Faz parte não conseguir fazer um ou outro exercício, mas você volta mais preparado(a) e FAZ!
