Termos Técnicos

Início >> painel >> termos técnicos

Relembre o significado de alguns termos técnicos na programação.

Algumas definições de termos técnicos:

Variável:
Uma variável é espaço na memória que armazena um valor, como x = 5.
Condicional:
Uma estrutura condicional permite executar ações diferentes dependendo de uma condição. Exemplo: if x > 10:
Laço de repetição:
Um laço de repetição Executa um bloco de código várias vezes. Exemplo: for i in range(5):
Função:
Uma função executa um bloco de código quando chamado é pode receber valores para trabalhar(parâmetros). Exemplo: function somar( param ) { ... }
